**CYBER PINGER ⚡️🚀** is a professional, beautifully designed network monitoring tool built natively as a Chrome browser extension. It allows you to track real-time connection stability, live Mbps speeds, and packet drops directly from your browser's toolbar—without ever switching windows!

---

## 🔥 Key Features

- **⚡ Live Data Visualization**: Features a dynamic, custom HTML5 Canvas line chart tracking your real-time **Mbps** network speed!
- **🌐 Smart Target Selection**: Connect to multiple default network portals or inject a custom target URL on the fly.
- **🛡️ Secure & Premium Interface**: Designed with a premium hacker-style "crack" theme, featuring neon glows, dark mode, and strict right-click/drag blocking.
- **🔄 Silent Background Execution**: Relies on Chrome's robust Manifest V3 Service Workers for reliable backend ping loops.
- **⚡ Cinematic Animations**: Built-in lightning strike animations visually confirm connection engagements.

## 🛠️ Technology Stack

- **Frontend Core**: HTML5, Advanced Vanilla CSS3 (Glassmorphism & Neon Glows), Modern ES6+ JavaScript.
- **Backend Architecture**: Chrome Extension Manifest V3, Chrome Storage API, Alarms API, and Async Fetch API.
- **Assets Integration**: Dynamic canvas chart generation and Base64 integrated network logos.

## 📦 Installation Guide

1. **Download or Clone** this repository to your local machine.
2. Open **Google Chrome** and navigate to `chrome://extensions/`.
3. Enable the **"Developer mode"** toggle located in the top right corner.
4. Click the **"Load unpacked"** button and select the folder containing this code.
5. The beautiful **CYBER PINGER** icon will instantly appear in your browser's extensions menu!
